syllabus

A preliminary version of the syllabus is available here. The high-level objectives for this course are that students should be able to:

  • Understanding the role and structure of compilers, and its various phases
  • Constructing an unambiguous grammar for a programming language
  • Generating a lexer and parser using automatic tools
  • Constructing machines to recognize regular expressions (NFA, DFA) and grammars (LL and LR parsers)
  • Generating intermediate form from source code
  • Type checking and static analysis
  • Assembly/binary code generation
